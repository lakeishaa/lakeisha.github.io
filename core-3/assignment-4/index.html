PREFACE


This book was created by computers.

Exploring the relationship between human relations and technology, this project uses Artificial Intelligence to generate content and is designed with code by a Javascript library called Bindery.js. In this experiment, I started merging the faces of the people that I talk to, slowly creating an ever expanding ammalgamation of the average faces of people in my daily life. Each generative content was created by a machine learning program called Artbreeder in an attempt to spark discourse on visualizing identity and change through the growing eminence of Artificial Intelligence.
